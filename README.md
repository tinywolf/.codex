## [Codex CLI](https://developers.openai.com/codex/cli/)
터미널에서 로컬 코드베이스를 읽고 수정하며 명령까지 실행할 수 있는 오픈소스 코딩 에이전트입니다.

## [Codex App](https://openai.com/ko-KR/index/introducing-the-codex-app/)
여러 에이전트를 병렬로 관리하고 워크트리 기반으로 장시간 작업을 조율할 수 있는 데스크톱 커맨드 센터입니다.

## [Codex SDK](https://developers.openai.com/codex/sdk)
Codex 에이전트를 코드에서 직접 제어해 CI/CD, 내부 도구, 애플리케이션 워크플로에 통합할 수 있는 SDK입니다.
