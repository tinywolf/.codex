## 수정 가이드

1. $1 이후 커밋된 내용을 참고해서, 각 문서의 목적을 고려하여 문서의 내용 중 수정 또는 삭제할 부분이 있는지 확인하세요.
2. 코드에서 삭제되어 문서 내에서 더 이상 유효하지 않은 내용이 있다면 내용을 삭제하고, "삭제되었음" 과 같은 언급은 하지 마세요.
3. 코드가 수정되어 문서 내 내용과 다르거나 추가해야 할 내용이 있다면 문서 내용을 수정하세요.
4. 수정 또는 삭제할 내용이 없다면 작업을 종료하세요.
