현재 브랜치에서 수정된 내용을 바탕으로 pull request 를 추가하세요. base 브랜치는 master 또는 main 입니다.
Pull Request 작성 시 `.github/pull_request_template.md` 존재 여부를 확인하고, 있다면 템플릿 형식을 준수해야 합니다.
내용은 한국어로 작성하세요.
Pull Request 내용을 작성할 때 링크를 추가해야 하는 경우, 변경 또는 추가된 주석을 참고하여 링크를 추출하여 사용합니다.
작성한 Pull Request 는 미리 보여주고, 반드시 승인을 거친 후 생성해야 합니다.
